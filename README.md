# Frontend

This folder contains the static Clinical Data Management dashboard UI.

## Contents

- `index.html`: Main page and all screen containers
- `styles.css`: Visual styling and animations
- `script.js`: Navigation, rendering, interactions, automation loop
- `data.js`: Generated data payload consumed by the UI

## Run

1. Generate fresh `data.js` from the pipeline:

```bash
cd ..\pipeline
python run_pipeline.py
```

2. Open the UI directly in a browser:

- Open `index.html` in Chrome or Firefox

No Node, npm, build tools, or frontend server are required.

## Data Source

- `data.js` is generated by:
  - `..\pipeline\export_to_frontend.py`
- The frontend reads `APP_*` constants from `data.js`.

## Notes

- The UI includes a 5-second synthetic automation loop for presentation flow.
- Lock mode pauses modification actions and keeps screens in read-only behavior.
